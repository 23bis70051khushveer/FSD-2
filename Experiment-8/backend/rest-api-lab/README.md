# Experiment 8: Develop RESTful APIs using Backend Framework

**UID:** 23BIS70051
**Class:** 23BIS-1(B)
**Name:** Khushveer Singh

---

## Learning Outcomes

1. **Understand RESTful API Architecture** — Gained knowledge of REST principles including HTTP methods (GET, POST, PUT, DELETE), status codes, and request/response lifecycle for building scalable backend services.

2. **Build Backend Server using Flask Framework** — Learned how to set up a Python Flask server, use Flask Blueprints for modular code organization, and create well-structured CRUD API endpoints.

3. **Test APIs using Postman** — Developed skills in using Postman to send HTTP requests, validate API responses, and debug backend endpoints during development.

4. **Deploy Backend Application on Cloud Platform** — Learned the process of deploying a Flask application on Render using Gunicorn as a production WSGI server, making the API publicly accessible.
