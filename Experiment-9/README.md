## 23BIS70051
# Authentication Experiment-9

## Learning Outcomes

By the end of this experiment, participants will be able to:

1. **Understand Authentication Mechanisms**: Differentiate between various web authentication strategies, including Basic Authentication, Custom Token-based Authentication, and JSON Web Tokens (JWT), understanding their respective use cases and security implications.

2. **Implement Secure Backend APIs**: Design and develop a Python (Flask) backend server capable of handling user logins, generating secure tokens, and protecting specific routes/endpoints using different authorization headers.

3. **Test APIs using Postman**: Effectively utilize Postman to construct HTTP requests, manipulate headers (such as `Authorization: Basic`, custom `x-auth-token`, and `Authorization: Bearer`), and validate successful and unauthorized responses from protected API routes.

4. **Deploy Web Services to the Cloud**: Successfully configure and deploy a containerized or managed Python backend application to a cloud hosting platform (Render), making the authentication API accessible over the public internet.
