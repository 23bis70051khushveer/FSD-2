## 23BIS70051

## 23BIS-1(B)

## Learning Outcomes

- Form Construction with MUI Components
  Learners will be able to design and structure interactive forms using Material UI (MUI) components such as TextFields, Checkboxes, Radio Buttons, and Buttons, following consistent design principles.

- Validation and State Management
  Learners will understand how to implement built-in and custom form validations (required fields, input formats, conditional validation) and manage form state effectively using React hooks.

- User Experience and Accessibility Awareness
  Learners will gain the ability to enhance usability and accessibility by applying proper labels, error messages, helper text, and feedback mechanisms aligned with Material Design standards.
